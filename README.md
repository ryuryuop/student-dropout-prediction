# Student Dropout Prediction using Machine Learning

This project predicts whether a university student will drop out or graduate using machine learning models.

## Models Used
- Logistic Regression
- Random Forest
- Gradient Boosting
- XGBoost

## Evaluation Metrics
- Accuracy
- Confusion Matrix
- ROC Curve
- Precision–Recall Curve
- Cross Validation

## Result
The models achieved around 94% accuracy in predicting student dropout.
